# svg-icons

SVG icon definitions for use with `@amag/icon-library`.

## Usage

```ts
import { AmagIconModule } from '@amag/icon-library';
import { amTrash, amSearch } from '@amag/svg-icons';
```

## Generating icons

Place SVG files in `raw-svgs/` then run:

```
nx run svg-icons:generate-icons
```
